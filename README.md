# spotify_clone

Spotify web clone with a Node.js/Express backend, MongoDB, and a Vue frontend.

## Project structure

- `server/` - Express API with MongoDB (Mongoose)
- `client/` - Vue 3 frontend (Vite)

## Backend setup

```bash
cd server
npm install
cp .env.example .env
npm run dev
```

## Frontend setup

```bash
cd client
npm install
npm run dev
```
